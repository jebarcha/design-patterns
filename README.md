## Creational

* Builder
* Factory Method
* Prototype
* Singleton

## Structural

* Composite
* Proxy
* Fly Weight
* Facade
* Bridge
* Decorator

## Behavioral

* Mediator
* Chain Of Responsibility
* Observer
* Strategy
* Command
* State
* Visitor
* Iterator
* Interpreter
* Memento
